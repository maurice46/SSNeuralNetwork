I implemented a neural network from scratch, including forward propagation, manual backpropagation, 
and gradient-based optimization without deep learning frameworks.

I designed a custom activation function to address vanishing gradient and exploding gradient issues 
that current activation functions have (ie. Hyperbolic Tangent Function and Rectified Linear Unit).

I then conducted and controlled experiments on real datasets (XOR and Wine Quality), analyzing 
training losses, accuracies, and generalization behavior across different learning rates.
